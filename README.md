### ----- Features -----

	• Facial Recognition: If face is recognized as an owner, unlock door and say "Welcome {Owner}". Or allow an option to say "Unlock door" with speech recognition.

	• Door lock: If not recognized, be able to use a keypad to still allow entry through the lock. Example, if you have a friend coming over who isn't recognized on the facial recognition, they'll still be able to unlock door with the keypad.
	
	• Keypad: to allow entry incase face isn't recongnized.
	
	• Door sensor magnets: to tell if door is closed or has been opened.

	• Light/motion sensor: When motion is detected, camera and light come on (Light comes on regardless of night or day, but helps when ever it's night to be able to see faces better. Camera stays on for 10 seconds, but resets every time new motion is detected.

	• Speaker: for text to speech
	
	• Buzzer: for alarm incase of forced entry

	• Website: to display all of the data with an admin panel.

	• Database: use a database to store all data. Pictures of unknown person plus data and time.

If camera does not recognize person, take a picture and email that picture to the owner/s. It'll be a silent alarm that goes off, but they can still access the keypad to unlock door. If door is forced entry, lock isn't unlocked by keypad or facial recognition, set an actual alarm with the buzzer for 30 seconds
